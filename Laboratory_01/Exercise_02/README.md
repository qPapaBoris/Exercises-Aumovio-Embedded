Create the circuit and the corresponding code for a traffic light system.

Using an Arduino board, three LEDs (Red, Yellow, Green), and one push button, implement a traffic light that operates as follows:

When the button is pressed, the traffic light starts.

The red LED stays on for 5 seconds.

Then the yellow LED turns on for 3 seconds.

After 3 seconds, the yellow LED turns off and the green LED turns on for 4 seconds.

After 4 seconds, the green LED turns off and the yellow LED turns on again for 3 seconds.

After another 3 seconds, the yellow LED turns off, the red LED turns on, and the program continues execution in a loop.
