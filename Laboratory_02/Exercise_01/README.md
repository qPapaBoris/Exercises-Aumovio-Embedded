Create a circuit consisting of two LEDs connected to pins 2 and 3, and three push buttons connected to pins 10, 11, and 12.

wokwi link: https://wokwi.com/projects/459821406136390657

The requirements are as follows:

Pressing the left button should turn on the left LED.
Pressing the right button should turn on the right LED.
Pressing the middle button should turn on both LEDs.

The implementation must be done using direct register manipulation only (do not use predefined functions such as digitalWrite()).

Part two: After completing the initial implementation, modify the code to use a separate header file containing all macros, definitions, and function prototypes.
