Task: Dual LED and Button Control, using Arduino Uno (ATmega328P)

wokwi link: 

Objective
    Create a circuit using two LEDs and two push-buttons. Each pair (LED and button) must operate independently following a specific logic.

Functional Requirements 
    Active State: While a button is pressed, its corresponding LED must blink continuously (toggle between ON and OFF states).
    Idle State: When the button is released, the LED must remain OFF.
    Independent Control: The behavior of the second LED and button pair must mirror the first one, operating separately.

Testing and Observation
    Perform the experiment using two different timing intervals to observe the change in visual behavior:
    Initial delay: 300 ms
    Fast delay: 50 ms

Hardware Components
2x LEDs, 2x Push-buttons, 2x resistors.
