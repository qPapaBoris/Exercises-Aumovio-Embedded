Two smart traffic lights are given. They are able to count both road traffic and pedestrian traffic and adjust their behavior based on the current context. The system must analyze collected data such as the number of vehicles and pedestrians in the intersection.

The implementation uses 6 LEDs and 3 push buttons to control the intersection.

Traffic light operating modes, to be used in later labs:
Day mode, Night mode, Low traffic, High traffic, Blinking mode, Default mode.

A custom AVR library, header file, will be developed for configuration.

The solution must be implemented using direct register manipulation for hardware configuration.

Requirement 1: Configure the required ports.
